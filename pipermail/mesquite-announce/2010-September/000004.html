Mesquite 2.1.2, a bug fix release in the Mesquite 2.1 series, is now
available for download.  This release primarily consists of fixes for
building on MacOS and testing a configured iMesh implementation.

Changes since 2.1.1:
 - Prefer 'c++' over 'CC' for choice of C++ compiler because 'CC' is
   not a C++ compiler on MacOS
 - Fix configure bug preventing enabling of tests for configured
   iMesh implementation
 - Fix minor build issues for tests of configured iMesh implementation
 - Fix broken support for iGeom/iRel APIs
 - Fix broken slave flag support in iMesh adaptor
