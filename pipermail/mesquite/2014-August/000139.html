Hi, Martin,

Mesquite just move mesh vertices, and don't modify the mesh topology. So 
after optimization, you still can convert it back to structure format.  
You can put the structure information into the order of the vertices, 
namely, you can order the unstructured mesh vertices from left to right 
and  bottom to up .

Best

Huayi

> Mesquite can be used to improve structured grid. You can convert 
> structured grid into unstructured format in vtk file, like following 
> (two quad):
>
> 3-------4-------5
> |        |         |
> |        |         |
> 0-------1-------2
> ```
> # vtk DataFile Version 3.0
> CGAL mesh
> ASCII
> DATASET UNSTRUCTURED_GRID
> POINTS 6 double
> 0.0 0.0
> 1.0 0.0
> 2.0 0.0
> 0.0 1.0
> 1.0 1.0
> 2.0 1.0
> CELLS 2 10
> 4 0 1 4 3
> 4 1 2 5 4
> POINT_DATA 6
> SCALARS fixed float
> LOOKUP_TABLE default
> 1
> 1
> 1
> 0
> 0
> 0
> ```
