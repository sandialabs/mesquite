David,

  Assuming your platform is Linux:

To configure the running of the unit tests you need a 3rd-party open
source unit testing framework called CppUnit that can be found at:

         http://sourceforge.net/projects/cppunit/files/cppunit/1.12.1/cppunit-1.12.1.tar.gz/download

Cppunit needs to be built and installed before it can be used.  Follow
the instructions included with the download above.


Configure Mesquite to include Cppunit:

    In CMake, search for 'cppunit' and:

          set CPPUNIT_INCLUDES to '<cppunit_install_dir>/include'
          set CPPUNIT_LIBRARY to '<cppunit_install_dir>/lib/libcppunit.so'.

  Then just Configure and Generate.  After making Mesquite, the ctest
command should run all the tests including the unit ones.

  For completion, if one was using GNU Autotools:

  From the top-level mesquite directory:
     $ ./configure --with-cppunit=<cppunit_install_dir>
     $ make
     $ make check


  If you have any problems or questions, let us know.
