On 02/06/2011 08:27 PM, Jibum Kim wrote:
> Hi,
>  
> I'd like to modify W matrix of Inverse Mean Ratio (IMR) quailty metric
> in Mesquite.
>  
> I think that the default value of W matrix is [ 1 0.5;0 sqrt(3)/2] which
> is for equilateral triangles.
>  

For src/QualityMetric/Shape/IdealWeightInverseMeanRatio.cpp that is the
default for triangle elements.  Obviously other defaults are used for other
element topologies.

> I noticed two invese mean ratio quality metrics implemented in Mesquite.
>  
> One is /src/QualityMetric/Shape/IdealWeightInverseMeanRatio.cpp and
> another is /src/QualityMetric/TMP/2D/InverseMeanRatio2D.cpp
>  
> What is the best way to modify W matrix of IMR metric nad put new values
> into W matrix?
>  

src/QualityMetric/TMP/2D/InverseMeanRatio2D.cpp does not have a default W.
The W matrix is obtained from the TargetCalculator instance.  If you create
a custom subclass of TargetCalculator you can have it generate whatever
values you want for W.
