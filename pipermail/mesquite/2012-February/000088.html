Oliver,

  The fix is indeed easy.  Just add the include for <cstddef> to src/include/Mesquite.hpp  
I haven't tested this on gcc 4.6 because I'm running CentOS 6 that does not currently support 4.6 but you shouldn't have any problem.

  We will add the change to the Mesquite source once we can test it with 4.6.
