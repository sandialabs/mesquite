This is a good discussion to have now.  I also have (local, uncommitted)
code that constrains 2D mesh nodes to 1-D interfaces and 3D mesh nodes to
2D interfaces.  We've been investigating imposing additional constraints
on the boundary constrained motion such as local volume conservation but
if basic functionality is needed now, we could commit ours and Allen's
changes and go from there.

Does Allen have a paper describing his method?
