Ok, this is where the root of some ideas are. I have two types of applications in mind. The first is one where I have a surface geometry specified by NURBS, probably IGES and possibly a CAD program such as AML, and a topology for a grid. In this mode, I would want MESQUITE to attach the topology to the geometry of the vehicle, where I would specify which nodes attach to which set of surfaces. For example, nodes that attach to the wall of the configuration and those that attach to an outer flow domain surface.

The second application is one where I have spent the time generating the wall grid of a vehicle, thereby defining the topology. I'd simply then extend the block corners to an outer boundary defined by facets (STL or triangular mesh). Then, let MESQUTIE move the nodes attached to the outer boundary as needed to get an "optimum" mesh.
